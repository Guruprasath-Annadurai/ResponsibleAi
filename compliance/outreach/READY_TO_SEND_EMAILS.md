## Before sending any of these

1. Replace every `[bracketed placeholder]` with real specifics — a
   generic mass email performs worse than a smaller number of specific,
   researched ones.
2. Find a real named contact wherever possible (LinkedIn, a company's own
   "team" page, a mutual connection) rather than a generic `info@` or
   contact-form address.
3. Send these yourself, from your own email address — this file is
   content, not a mail-merge tool.
